# Decision Tree (SMOTE)

![Untitled](Decision%20Tree%20(SMOTE)%20b9ea0181dc1e4106871f5191757a85ca/Untitled.png)

특정 기준에 따라 데이터를 구분하는 모델. 

특정 기준은 데이터 변수를 뜻함. 

변수의 중요도가 높을수록 제일 먼저 데이터를 구분하는데에 사용.
