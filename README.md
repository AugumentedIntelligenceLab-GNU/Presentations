# 💁 증강지능 연구실 소개 

## 증강지능이란?

- Augmented Intelligence

### 정의

- 인공지능(AI)과 인간의 지능의 결합

- 인간이 인공지능의 도움을 받아
    - 인지능력 확장
    - 의사결정 도움
    - 문제해결능력 향상

### 목표 ⭐

- 인공지능을 인간의 가치판단과 의사결정을 지원하는 기술로 발전시키는 것

- 인간의 지능과 기술을 보완
  - 인간의 지능을 복제하는 것을 목표로 하는 인공지능과 다름
  
- 인공지능에 자율성 부여는 지양
