# 💁 증강지능 연구실 소개 

## 증강지능이란?

- Augmented Intelligence

### 정의

- 인공지능 + 인간
- 인공지능의 도움을 받아 인간의 인지능력이 확장

### 목표 ⭐

- 인공지능을 인간의 가치판단과 의사결정을 지원하는 기술로 발전시키는 것
  - 인공지능에 자율성 부여는 지양
